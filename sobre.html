O que é o Gerador de Imagens com IA?

Este projeto é uma representação utilizando html e CSS para mostrar como seria feito o FrontEnd de um gerador de imagens ultilizando Inteligência Artifical. A página galeria, seria usada como uma espécie de armazém para imagens geradas e a seção de gerar imagem, é a página que seria pedido um promt para gerar a imagem.
